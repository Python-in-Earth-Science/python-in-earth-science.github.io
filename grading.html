Assignment details

Warm-up problems (TopHat)

A short coding question is posed at the start of each lecture via TopHat. Completing it within the first few minutes counts — responses are timestamped. This rewards showing up on time and being ready to work. There is no penalty for an incorrect answer, only for not responding. Join TopHat at tophat.com using your Vanderbilt email and the course join code provided in class.

Coding assignments

Longer take-home problems that ask you to apply the week's concepts to a new geoscience problem. Submitted via Brightspace in weeks 1–5 and via GitHub Classroom from week 6 onward.

Labs

The Monday afternoon lab period (1:25–4:15pm) is the core of the course. Each week you'll work through an applied problem using that week's tools and real Earth science data. Labs are due the following Sunday at 11:59pm. Submitted via Brightspace in weeks 1–5 and via GitHub Classroom from week 6 onward. There are 11 graded labs this semester (Week 1 is an ungraded orientation; Labor Day and Thanksgiving reduce the total from 15).

Peer feedback workshop (Week 13 lab)

A structured in-class review of final project proposals. Your final project proposal must be submitted before arriving. Attendance and substantive written feedback are required for full credit.

Final coding project

An independent project in which you use Python to investigate a geoscience question of your choosing using real data. Submitted via GitHub Classroom. Due date announced during Week 13. More details distributed mid-semester.

Python package presentation Graduate only

Each graduate student presents a Python package of their choice (15–20 minutes) at the start of a lab period during weeks 7–11. The package should be relevant to your research area. Presentations are scheduled in Week 6 — sign up for your slot then.

Git skills assignment Graduate only

A short practical assignment covering branching, merging, and pull requests. Due Week 15. Details distributed in Week 6 alongside the Git lecture content.

Policies

Late work: Labs and assignments lose 10% per day late unless you have contacted Dr. Morey in advance. The peer feedback workshop and final project cannot be submitted late.
Collaboration: You are encouraged to discuss problems with classmates, but all code you submit must be written by you. Do not share notebook files or copy code directly from another student's submission.
AI tools: You may use AI coding assistants (GitHub Copilot, ChatGPT, etc.) as a learning aid, but you must be able to explain every line of code you submit. If you cannot explain it, it does not count as your work.
