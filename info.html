Course description

EES 3350 & EES 5350: Introduction to Python in Earth Science (4 credit hours)

Introduction to scientific programming using Python for earth and environmental science applications. Covers programming fundamentals, scientific computing packages, and geospatial analysis. Students apply Python skills to solve real-world problems including climate data analysis, surface process modeling, and geospatial data visualization.

All coursework is conducted using Jupyter Notebooks, which allow for interactive coding, documentation, and visualization in a single environment. The course is organized into four units: Python & Scientific Computing Basics, Scientific Python Packages, Geospatial Analysis, and Modeling & Advanced Topics. Starting in Week 6, students use Git and GitHub for version control and assignment submission — a skill that translates directly to scientific research workflows.

Learning outcomes

Upon successful completion of EES 3350/5350, students will be able to:

  • Program effectively in Python — write well-structured, documented Python programs to solve scientific problems including data input/output, mathematical calculations, and basic modeling
  • Apply core programming concepts — demonstrate understanding of variables, data types, functions, control structures, and debugging strategies
  • Utilize scientific computing packages — effectively use NumPy, pandas, Matplotlib, SciPy, seaborn, and GeoPandas for data manipulation, analysis, and visualization
  • Create publication-quality figures — customize plots with appropriate axes, annotations, colorbars, and styles suitable for scientific communication
  • Solve geoscientific problems — apply programming skills to earth science challenges including analyzing climate data, modeling erosion and diffusion, processing geospatial and raster datasets, and working with real field measurements
  • Conduct geospatial analysis — perform vector and raster geospatial data analysis and visualization using GeoPandas, rasterio, and related tools
  • Use version control — manage code with Git and GitHub, submit work via GitHub Classroom, and understand the basics of collaborative scientific software development
  • Interpret and modify existing code — read, understand, and modify scientific Python scripts written by others
  • Create professional scientific documentation — develop well-documented notebooks that combine narrative, code, and results, building a portfolio of programming examples for future reference
